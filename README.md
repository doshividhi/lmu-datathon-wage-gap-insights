# 🎓 LMU Datathon — Gender Wage Gap Dashboard

> A data visualization project built during the **LMU Datathon**

## 👉 [Click here to view the Live Dashboard](https://doshividhi.github.io/lmu-datathon-wage-gap-insights/)
### About
An interactive dashboard exploring gender wage gap trends using CPS data, 
analyzing the impact of California's 2018 Salary History Ban on wage growth 
across gender and region.

### Key Insights
- 📈 Female wage growth overtook male for the first time post-ban (~29% vs ~27%)
- 💰 Wage gap persists at ~14% despite both genders growing significantly
- 🌎 Western states narrowed the gap faster after the 2018 ban
